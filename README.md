# Kanban/Trello Board

## Built with 
- Reactjs
- Apollo Client Graphql
- CSS
- React Hooks

## Features
- Create Columns
- Create Tasks in each Column
- Update Tasks
- Delete Tasks
- Move Tasks from one column to another
- Sort by createdDate
